Title: TaskFlow
TaskFlow is a simple, calm task manager designed for anyone. Students, workers, or home users, who wants their day to feel more organized and less stressful. With TaskFlow, you can quickly view tasks, track progress, and see upcoming deadlines in a clean, minimal dashboard without distractions.

Features:
Dashboard with task KPIs (total, completed, in-progress, overdue)
Task status bar chart using Chart.js
Task list page with priorities and statuses
Light/dark mode toggle with saved preference
Simple login “demo” flow (front-end only)
Profile dropdown with logout
Responsive layout for desktop and smaller screens

Frameworks / Libraries Used:
Bootstrap 5.3 – layout, grid, and basic components
Chart.js – bar chart on dashboard
Font Awesome 6 – icons (moon/sun, logout, etc.)
Google Fonts (Roboto, Caveat) – typography
Vanilla JavaScript – login demo logic, theme toggle, and chart setup
